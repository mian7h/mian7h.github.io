◆ Program Learning Outcomes

What This Program Taught Me

The MS in Applied Data Science is built around six core competencies. Each of my projects maps directly to one or more of these outcomes.

01

Collect, Store & Access Data

Identifying and leveraging applicable technologies for data acquisition and management at scale.

02

Create Actionable Insight

Generating insight across societal, business, and political contexts using the full data science lifecycle.

03

Visualization & Predictive Models

Applying visualization and machine learning to help stakeholders make better, faster decisions.

04

Programming Languages

Using R and Python to support the generation of insight from complex, large-scale datasets.

05

Communicate Insights

Translating technical findings into clear narratives for both technical and non-technical audiences.

06

Apply Ethics

Evaluating data and predictive models for fairness, bias, transparency, and privacy implications.
